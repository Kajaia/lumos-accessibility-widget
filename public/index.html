Welcome to the test page for the Lumos Accessibility Widget. This page demonstrates how the widget works on a real website with text and images.

The widget provides accessibility tools to help all users navigate your website easily. You can try different options by interacting with the toolbar. Learn more.

You can customize the widget for your own site, add features, and ensure it works smoothly with your content. For example, adding a screen reader can enhance usability for visually impaired users.

Features Overview

The Lumos Accessibility Widget includes various tools, such as high-contrast mode, text resizing, and a screen reader. Each tool can be activated independently to improve user experience.

These features are designed to meet accessibility standards and help create an inclusive online environment for all visitors.

  • High-contrast mode
  • Text resizing
  • Screen reader (Text-to-Speech)

Additional settings allow you to adjust colors, fonts, and layout preferences. You can also provide quick links to important content for your users.

Getting Started

To start using the widget, include the JavaScript file in your website and initialize it. You can also customize the toolbar and feature options to match your site’s design.

Usage Example

After adding the widget, users can click the toolbar buttons to enable accessibility features instantly. You can also programmatically control features using the provided API.

The widget is responsive and works on all devices, including desktops, tablets, and mobile phones.

Our goal is to ensure every visitor can access your content easily, regardless of their abilities or device.

If you encounter any issues or have questions, please refer to the documentation or contact support for assistance.

Advanced Settings

The widget allows advanced users to configure features such as keyboard navigation, screen reader voices, and custom hotkeys.

Proper configuration ensures that the accessibility tools work smoothly with your existing website layout and content structure.
